Particles

Initialize plugin in your js file.

    
    new Particles ({
        direction: true,     // automatic reversible
        canvas: '#particles',       // your canvas
        width:  window.innerWidth,  // canvas width
        height: window.innerHeight, // canvas height
        color: '#2c3e50',       // dots color
        speed: 100,             // dots speed
        dots: 150,              // dots count
        rmin: 2,                // minimum radius
        rmax: 7,                // maximum radius
        bg: '#fff'              // canvas background
    })
